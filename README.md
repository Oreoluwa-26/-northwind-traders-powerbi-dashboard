# Northwind Traders — Executive BI Dashboard

## Project Overview
A business intelligence dashboard built in Power BI for Northwind Traders, 
a global gourmet food supplier. The dashboard gives executive leadership a 
clear view of sales performance, product intelligence, and regional & people 
performance.

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- CSV datasets (8 tables)

## Dashboard Pages
- **Page 1** — Sales & Revenue Overview
- **Page 2** — Product & Category Intelligence
- **Page 3** — Regional, Operational & People Performance

## Key Insights
- Total Revenue: $1.27 Million across 830 orders
- Beverages is the #1 revenue category
- USA and Germany are the top markets
- Côte de Blaye is the single highest revenue product at ~$150K
- Margaret Peacock is the top performing sales employee

## Files Included
- `northwind_dashboard.pbix` — Power BI dashboard file
- `northwind_report.pdf` — Full business intelligence report
- `datasets/` — All 8 CSV source files

## How to View the Dashboard
Download the .pbix file and open it in Power BI Desktop (free to download 
from microsoft.com). All data is already embedded in the file.
